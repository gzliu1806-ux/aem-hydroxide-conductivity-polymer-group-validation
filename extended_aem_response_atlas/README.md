# Full AEM response atlas extension

GitHub extension candidate; not part of the formal Supporting Information.

This atlas records the 48 eligible continuous response profiles from the declared 78-variable AEM descriptor block. The 12 landscape pages contain four panels each. Curves use the same reference CatBoost calculation as Fig. 3 and Fig. S6: one descriptor is varied across 55 points between its empirical 5th and 95th percentiles, other selected inputs are held at training-set medians, and the light-blue background is the normalized empirical support histogram.

Small-amplitude or nearly flat fitted responses are retained because the atlas is a complete diagnostic, not a value ranking. A flat conditional profile does not show that a descriptor has no scientific value; it only shows limited marginal response in this fitted model over the sampled range. The formal manuscript uses two representative profiles and the Supporting Information uses eight selected profiles.
